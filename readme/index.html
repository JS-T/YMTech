协议

本说明书中各项接口均采用 HTTPS 协议

服务器状态

GET /status/alive/

服务器运行正常

{
    "status": 200, 
    "timestamp": 1471842394, 
    "message": "Hello world!"
}

获取来源列表

GET /status/origin/

来源列表

{
    "status": 200, 
    "origin": [
        {
            "host": "127.0.0.1", 
            "name": "Localhost", 
            "power": 1, 
            "id": 1
